## Editors Notes

1. Ensure you are using the LTS version of Node.js (`nvm install lts/* --reinstall-packages-from=node`)
2. Clone the repo
3. `npm install`
4. `gulp`
5. Visit `http://localhost:4000/EditorsDraft/index.html` which will reload on changes
